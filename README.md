# Simple-absent-apps-with-pyqt5

# Dependencies

- python 3.9.13
- pyqt5
- sqlite3

# Installation

1. Install the pyqt5 using pip

```
pip install pyqt5
```

2. Install pyqt5-tools using pip

```
pip install pyqt5-tools
```

3. Install pyqt5 Extension on VSCode
